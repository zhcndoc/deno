禁止比较两个完全相同的值或变量。

将一个变量或值与其自身进行比较通常是一个错误，可能是拼写错误或重构错误。这对读者来说可能会造成困惑，并可能引入运行时错误。

**无效：**

```typescript
if (x === x) {
}
if ("x" === "x") {
}
if (a.b === a.b) {
}
if (a["b"] === a["b"]) {
}
```

**有效：**

```typescript
if (x === y) {
}
if ("x" === "y") {
}
if (a.b === a.c) {
}
if (a["b"] === a["c"]) {
}
```
