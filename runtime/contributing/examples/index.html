贡献一个示例

Deno 示例 是一个展示如何使用 Deno 及其 API 的示例集合。如果您贡献一个示例，我们将送您一包免费的贴纸！

贡献一个示例 Jump to heading#

如果您有一个 Deno 示例想要与社区分享，您可以将其贡献到 Deno 文档库 或者提出一个问题，告诉我们您希望看到的示例。如果您的示例被合并，我们将以作者的名义给您致谢，并送您一些很棒的特别版 Deno 贴纸，以展示您作为贡献者的身份，作为我们感谢的象征。

获取您的贴纸 Jump to heading#

如果您贡献了一个示例，请通过电子邮件与我们联系，发至 docs@deno.com，告诉我们您以便我们能够将贴纸寄给您！
